批量添加单词

选择"全部模式"将把单词添加到所有激活的模式中，或选择特定模式只更新该模式的数据

JSON格式示例:

[
  {"word": "vivid", "translations": ["生动的"]},
  {"word": "ambiguous", "translations": ["模糊的"]}
]

选择全部模式时，只需提供单词和翻译；选择特定模式时，可额外提供errors和practiceCount数据。

CSV格式示例:

vivid,生动的,0,0
ambiguous,模糊的,0,0

选择全部模式时，只需提供单词和翻译（前两列）；选择特定模式时，可额外提供错误次数和练习次数（后两列）。

Excel复制数据示例:

vivid    生动的    0    0
ambiguous    模糊的    0    0

选择全部模式时，只需提供单词和翻译（前两列）；选择特定模式时，可额外提供错误次数和练习次数（后两列）。
